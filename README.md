# To-Do-List
C# 윈도우폼으로 만든 TO DO LIST 윈도우 전용 프로그램입니다. 
# 🗒️ To Do List


Tools : Visual Studio 2019

Language : C#


- 간단하게 사용하려고 만든 To-Do-List 입니다.
- 로컬에서 할 일을 저장한 걸 볼 수 있습니다.
- txt 파일 형식으로 문자열을 로컬에 저장하고 읽어옵니다.
- 할 일을 추가, 삭제, 완료, 업데이트, 저장, 업로드를 할 수 있습니다.

## 💫 Stack & Framework


- C#
- Windows Forms
